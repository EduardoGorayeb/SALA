Vamos para o aqui, para explicar o que é clonagem. A clonagem é um processo que tem, né, na ciência, que é de criar uma cópia idêntica a de organismo, célula ou molécula. O clone resultante tem a mesma composição genética do organismo de origem. Ele, em organismos multicelulares, o processo envolve a transferência do núcleo de uma célula somática, célula do corpo, para um óvulo cujo núcleo foi removido, criando assim um novo organismo com as mesmas características, como a gente vai mostrar que é o exemplo da ovelha Dolly, que foi criada com as mesmas características de uma ovelha que foi um clone. Aqui a gente tem os tipos de clonagens, que são três tipos, que é a clonagem natural, que ocorre quando tem gêmeos idênticos, quando nasce gêmeos idênticos da da mulher, né? Aí a gente tem a reprodutiva, cujo objetivo é criar um organismo inteiro geneticamente idêntico ao outro, e a terapêutica, que busca produzir células tronco para tratamento de doenças sem gerar um novo indivíduo. A de células troncos, ela é principalmente usada, que tá sendo usada em pessoas que têm diabetes. Elas são usadas para conseguir reconstruir aquelas pessoas que têm diabetes, que vão perder uma perna, um braço, a as a medicina usa para que essa pessoa não perca por meio dessas células tronco. Aí aqui a gente tem um exemplo da ovelha Dolly. Eu vou ler um pouquinho sobre a ovelha Dolly. Quem foi a ovelha Dolly? Ela foi o primeiro mamífero clonado a partir de células adultas. O núcleo de uma célula das glândulas mamárias de uma ovelha Finn Dorset, cabeça branca, foi transferido para um óvulo sem núcleo de uma Scottish Blackface, cabeça preta. A outra ovelha da mesma raça gerou a Dolly, idêntica a Finn Dorset. O uso das raças diferentes serviu para evitar confusão entre as características.
